Descriptions :
Ce projet est l'attente pour l'examen devOps 2026

Il contient une function de validation d'adresse mail en typescript
La validation contient :

- Mail avec un @ minimum
- Texte avant et aprés @
- Mail sans espace
- Un point dans le domaine du mail

Ces tests ont été écris comme vus au cours et au projet du Q1 , en TDD
chaque régle a été tester puis a été implémenté dans le code

Une pipeline actions github a été mise en place afin d'éxecuter automatiquement les tests lors d'un changement dans le main/master (dans ce cas main)


Et finalement pour éxecuter le code vous devez lancer la commande dans le terminal : npm test
